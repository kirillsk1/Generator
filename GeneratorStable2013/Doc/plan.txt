Проблема 1. VarName, ParamName. Хочется скинуть и переменные и параметры в один список, но
поля у этих разных объектов разные, поэтому не получится единообразно к ним обращяться.
Хотелось бы Избежать нетерминала VarName, а только Name.
Например, вместо
Var := Type, VarName, ";\n"
VarName := NameCount("v")
Param := Type, ParamName
ParamName := NameCount("p")

писать
Var := Type, VarName, ";\n"
	Name := NameCount("v")
Param := Type, ParamName
	Name := NameCount("p")

То есть какая-то вложенная именовка нетерминалов, но это нереально.

Реально сделать псевдонимы для выражений.

Var := Type, VarName AS Name, ";\n"
VarName := NameCount("v")
Param := Type, ParamName AS Name
ParamName := NameCount("p")

или проще:
Var := Type, NameCount("v") AS Name, ";\n"
Param := Type, NameCount("p") AS Name
